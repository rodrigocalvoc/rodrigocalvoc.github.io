Comentarios en JavaScript

Los comentarios son líneas de código que JavaScript ignorará intencionalmente. Los comentarios son una excelente manera de dejar notas para ti mismo y para otras personas que necesitarán descubrir más tarde qué hace ese código.

Hay dos formas de escribir comentarios en JavaScript:

  • Usar // le dirá a JavaScript que ignore el resto del texto en la línea actual. Este es un comentario en línea:

    • // Este es un comentario en línea.

  • Puedes hacer un comentario de varias líneas comenzando con /* y terminando con */. Este es un comentario de varias líneas:

    • /* Este es un
    comentario de varias líneas */

NOTA: A medida que escribes código, debes agregar regularmente comentarios para aclarar la función de partes de tu código. Comentar bien puede ayudar a comunicar la intención de tu código, tanto para otros como para tu yo futuro.

Intenta crear un comentario de cada tipo.

// Hola Mundo

/* Hola a todos!!
¿Qué tal estáis? */
