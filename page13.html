Multiplicar dos números con JavaScript

También podemos multiplicar un número por otro.

JavaScript utiliza el * símbolo para la multiplicación de dos números.

Ejemplo:


const myVar = 13 * 13;

myVar tendría el valor 169.

Cambie el 0 para que el producto sea igual a 80.

        
        const difference = 8 * 0;
    
        const difference = 8 * 10;
