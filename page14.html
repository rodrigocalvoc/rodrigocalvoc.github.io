Dividir un número por otro con JavaScript

También podemos dividir un número por otro.

JavaScript utiliza el / símbolo para la división.

Ejemplo:


const myVar = 16 / 2;

myVar ahora tiene el valor 8.

Cambie el 0 para que quotient sea igual a 2.

        
        const quotient = 66 / 0;
    
        const quotient = 66 / 33;
