Incrementar un número con JavaScript

Puede incrementar o agregar uno fácilmente a una variable con el ++ operador.

i++; es el equivalente de

    
    i = i + 1;

Nota: La línea entera se convierte en i++;, eliminando la necesidad del signo igual.

Cambie el código para utilizar el ++ operador en myVar.

        
        let myVar = 87;

        // Only change code below this line
        myVar = myVar + 1;
    
        myVar++;
