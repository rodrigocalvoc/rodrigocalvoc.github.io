Sumar dos números con JavaScript

Number es un tipo de dato en JavaScript que representa datos numéricos.

Ahora intentemos sumar dos números usando JavaScript.

JavaScript utiliza el + símbolo como operador de suma cuando se coloca entre dos números.

Ejemplo:


const myVar = 5 + 10;

myVar ahora tiene el valor 15.

Cambie 0 para que la suma sea igual a 20.


        const sum = 10 + 0;
    
        const sum = 10 + 10;
